# Factory Coffee
Factory Coffee is a Progressive Web App (PWA) designed for seamless online coffee ordering and pickup management.

## Project Overview
This project provides a modern, fast, and responsive interface for customers to browse coffee products, manage their cart, and track orders. It also features a specialized interface for shop staff to manage order fulfillment via QR code scanning.

## Key Features
* **Modern UI**: Designed for speed and aesthetics using Bootstrap and AppKit components.
* **PWA Capability**: Can be installed on mobile devices for a native-like experience.
* **Interactive Ordering**: Dynamic cart system and product browsing.
* **Staff Portal**: Built-in QR scanner interface for verifying and filling customer orders.
* **Responsive Design**: Optimized for various screen sizes, including a custom landing page with motion effects.

## Technologies Used
* **Frontend**: HTML5, CSS3, JavaScript (jQuery), Bootstrap.
* **Architecture**: AJAX-based page transitions for a smooth, single-page app feel.
* **Version Control**: Git & GitHub.
